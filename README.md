# PowerPro Web ⚾

野球シミュレーションゲーム - Three.js + Google Apps Script

## 機能

- ✅ 3D野球グラウンド（4種カメラ固定 + 打球追従）
- ✅ 12種の変化球（ストレート・横変化・縦変化・チェンジアップ）
- ✅ 投手左右・打者左右設定
- ✅ 投手の球種割合設定
- ✅ マグナス効果を含む投球軌道計算

## 技術スタック

- **フロントエンド**: Three.js, HTML5, JavaScript
- **バックエンド**: Google Apps Script
- **データベース**: Google Sheets

## デプロイ方法

1. Google Apps Script エディタで Code.gs と Index.html を更新
2. デプロイ → 新しいデプロイ → ウェブアプリ で公開
3. ブラウザでアクセス

## 使い方

### チーム管理
1. **練習チーム作成** ボタンで初期チーム A/B を生成
2. チーム名を編集して保存

### 選手管理
1. **新規選手** で選手作成
2. 名前、ポジション、能力値を入力
3. 投手の場合：球種割合を設定（自動配分可能）
4. **選手保存** で保存

### 試合開始
1. ホーム・ビジターチームを選択
2. イニング数を指定
3. **⚾ ゲーム開始** をクリック

### カメラ操作
- **キャッチャー後方**：標準ビュー
- **マウンド前方**：投手目線
- **3塁側/1塁側**：スタンド視点
